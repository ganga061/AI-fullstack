DELIVERABLE 2: Live Demo Link
-------------------------------
Option A (Quick - no signup): Use localhost recording
  → Record video of http://localhost:5173 running locally

Option B (Free deploy in 10 mins):
  Backend → Railway.app:
    1. Go to https://railway.app
    2. "New Project" → "Deploy from GitHub"
    3. Select your AI-fullstack repo → select /backend folder
    4. Add environment variable: DATABASE_URL = file:./dev.db
    5. Copy the Railway URL (e.g. https://ai-fullstack-production.up.railway.app)

  Frontend → Vercel:
    1. Go to https://vercel.com
    2. "Add New Project" → Import AI-fullstack repo
    3. Set Root Directory to "frontend"
    4. Add env var: VITE_API_URL = [your Railway URL]
    5. Deploy → get live URL
